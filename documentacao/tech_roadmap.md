# Roadmap SaaS - StreamHub

Este roadmap foca na monetização e retenção de longo prazo.

## Fase 2.1: Experiência Premium (UX & Engajamento)
**Objetivo**: Fazer o usuário sentir que o app é "sofisticado".
- [ ] **UI Overhaul**: Adotar Glassmorphism, animações fluidas (Framer Motion) e Dark Mode OLED real.
- [ ] **Curadoria Editorial**: Seção de "Destaques da Semana" na Home (Manual + IA).
- [ ] **Feature**: Listas Compartilháveis (Compartilhar link dinâmico).

## Fase 2.2: O Motor de Assinaturas (Monetização)
**Objetivo**: Converter usuários Free em Pagantes.
- [ ] **Gatekeeping**: Bloquear features premium (ex: Estatísticas, Modo Aleatório) no frontend.
- [ ] **Pagamentos**: Integração com Stripe ou Pagar.me.
    -   Plano Mensal (R$ 14,90).
    -   Plano Anual (desconto).
- [ ] **Onboarding Premium**: "Teste Grátis por 7 dias".

## Fase 2.3: Inteligência Real (Retenção)
**Objetivo**: Entregar valor que justifique a mensalidade.
- [ ] **Tracker de Assinaturas**: CRUD manual de gastos ("Eu assino X, Y, Z").
- [ ] **Advanced Recommender**:
    -   Filtragem por Mood ("Quero rir", "Quero chorar").
    -   Filtragem por Tempo Disponível ("Tenho 20min").

## Fase 2.4: Expansão (Growth)
- [ ] **SEO Pages**: Gerar páginas estáticas `/melhores-filmes-netflix` via Next.js SSG.
- [ ] **App Nativo**: Voltar o foco para o React Native/Expo com base no sucesso da Web.
